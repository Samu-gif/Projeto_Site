A Web
Começa Aqui!

Aprenda HTML — a base de toda a web! Você vai descobrir como estruturar páginas com tags essenciais, criar links, listas, imagens e formulários, entender a semântica para sites bem organizados e praticar com exemplos reais.

CSS:
Cores, Layouts e Magia!

Aprenda a transformar sites simples em experiências incríveis com CSS! Você vai dominar cores, efeitos visuais, layouts modernos com Flexbox e Grid, animações fluidas e design responsivo.
CSS é a magia que dá vida ao visual da web — e você vai aprender todos os seus segredos!

JavaScript:
Dê Superpoderes aos Seus Sites!

Aprenda a transformar páginas estáticas em experiências interativas! Você vai criar funcionalidades dinâmicas, dominar lógica de programação, trabalhar com dados em tempo real e desenvolver mini-projetos práticos como calculadoras e jogos.
JavaScript é a linguagem que faz a web ser inteligente e interativa - e você vai começar a dominá-la agora mesmo.
